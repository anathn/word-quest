### Issue #1: Accuracy Calculation Incorrectly Uses `words_correct` Counter ✅ FIXED
- **Severity:** 🔴 Critical → ✅ Resolved
- **Location:** `src/components/session_tracker.py:49-52`
- **Problem Identified:** The `words_correct` counter was being incremented in `record_attempt()` but the property calculated from the `words` list, causing potential desync.
- **Resolution:** Removed the redundant counter increment logic from `record_attempt()`. The `words_correct` property now serves as the single source of truth, calculating from the `words` list which is always accurate.

```python
# Fixed: words_correct property calculates from words list (line 57-60)
@property
def words_correct(self) -> int:
    """Get count of words spelled correctly on first attempt."""
    return sum(1 for w in self.words if w.first_attempt_correct)
```

### Issue #2: Streak Tracking Logic Broken in `record_attempt()` ✅ FIXED
- **Severity:** 🔴 Critical → ✅ Resolved
- **Location:** `src/components/session_tracker.py:265-271`
- **Problem Identified:** The streak logic checked `self._word_attempts == 1` AFTER incrementing, causing it to never trigger.
- **Resolution:** Captured `is_first_attempt` state BEFORE incrementing `_word_attempts`, then used that variable for streak logic.

```python
# Fixed (lines 240-252):
def record_attempt(self, is_correct: bool):
    if not self._current_word:
        return
    
    # Track if first attempt was correct - capture BEFORE increment
    is_first_attempt = (self._word_attempts == 0)
    if is_first_attempt:
        self._word_first_attempt_correct = is_correct
    
    self._word_attempts += 1
    
    # Update last activity for idle timeout
    self._last_activity = datetime.now()
    
    # Track streak using is_first_attempt, not _word_attempts after increment
    if is_correct and is_first_attempt:
        self._current_streak += 1
    elif not is_correct:
        self._current_streak = 0
```

### Issue #10: No Session Timeout Definition ✅ FIXED
- **Severity:** 🟡 Medium → ✅ Resolved
- **Location:** `src/components/session_tracker.py`
- **Resolution:** Implemented idle timeout functionality (30 minutes default) with `check_idle_timeout()` method and automatic session completion.

```python
# Added (lines 186-195):
self.IDLE_TIMEOUT_SECONDS = 1800  # 30 minutes

def check_idle_timeout(self):
    """Check if idle timeout exceeded and auto-complete session."""
    if not self._last_activity or not self.current_session:
        return
    
    idle_time = (datetime.now() - self._last_activity).total_seconds()
    if idle_time > self.IDLE_TIMEOUT_SECONDS:
        import logging
        logging.info(f"Idle timeout ({idle_time:.0f}s > {self.IDLE_TIMEOUT_SECONDS}s), auto-completing session")
        self.complete_session()
```
