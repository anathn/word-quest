### 2. ✅ Particle Memory Leak

**Problem:** `BadgeNotification._create_particles()` created particles that were never cleaned up, potentially causing memory leaks.

**Solution:** 
- Added `cleanup()` method to `BadgeNotification` to clear particles
- Modified `update()` method to filter out dead particles each frame
- Added `_particles_created` flag to track particle state

**Code Changes (src/ui/badge_notification.py):**
```python
# Added particle cleanup in update()
self.particles = [p for p in self.particles if p.alpha > 0]

# Added cleanup method
def cleanup(self):
    """Clean up all particles to prevent memory leaks."""
    self.particles.clear()
    self._particles_created = False

# Added flag tracking
self._particles_created = True  # in __init__
```
