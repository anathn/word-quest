### Issue #4: PLANET_4 vs PLANET_5 Borderline Distinguishability

- **Severity:** 🟡 Medium
- **Location:** Theme color configuration
- **Problem:** Brown (121, 85, 72) and Gold (205, 170, 80) have relatively low RGB difference (84) and may appear similar to some color-blind users, especially in small UI elements.

- **Impact:** Users with severe color vision deficiencies might still confuse Planet 4 and Planet 5

- **Recommended Fix:** 
  1. Run additional validation:
     ```python
     validator = ColorValidator()
     brown = (121, 85, 72)
     gold = (205, 170, 80)
     
     passes, reason = validator.validate_color_pair_with_reason(brown, gold)
     if not passes:
         print(f"Not distinguishable: {reason}")
     
     # Check simulations
     deut_diff = validator.get_color_difference(
         validator.simulate_deuteranopia(brown),
         validator.simulate_deuteranopia(gold)
     )
     print(f"Deuteranopia distance: {deut_diff}")
     ```
  2. If distance < 80, consider alternative:
     - Option A: Make PLANET_5 more teal (0, 188, 212) for better contrast
     - Option B: Make PLANET_4 darker brown (90, 60, 50) for more contrast
     - Document the choice with validation results
