## Critical Issues (Must Fix Before Merge)

These issues block the merge and must be addressed:

### Issue #1: Milestone Re-Trigger Bug

- **Severity:** 🔴 Critical
- **Location:** `src/components/streak_bonus.py:check_milestone()` (lines 66-80)
- **Problem:** The milestone detection logic allows a bonus to trigger even when the streak has already surpassed the threshold. Calling `check_milestone(3)` after streak 4 has been reached will incorrectly trigger the 3-word bonus.
- **Impact:** Milestone bonuses can trigger even when the streak was missed, breaking the "once per session" guarantee and potentially triggering bonuses out of order.
- **Recommended Fix:** Track the previous streak value and only trigger milestones when the streak is incrementing through the threshold.

**Current Code:**
```python
def check_milestone(self, streak: int) -> Optional[BonusMilestone]:
    for milestone in self.milestones:
        if streak == milestone.streak_threshold and not milestone.triggered:
            milestone.triggered = True
            # ...
            return milestone
    return None
```

**Fixed Code:**
```python
def __init__(self):
    # ... existing init ...
    self._previous_streak = 0  # Add this line

def check_milestone(self, streak: int) -> Optional[BonusMilestone]:
    """Check if current streak triggered a bonus milestone."""
    for milestone in self.milestones:
        threshold = milestone.streak_threshold
        # Only trigger if we're incrementing THROUGH the threshold
        if (self._previous_streak < threshold <= streak) and not milestone.triggered:
            milestone.triggered = True
            self.active_bonus = milestone
            
            if self._on_bonus_triggered:
                self._on_bonus_triggered(milestone)
            
            self._previous_streak = streak
            return milestone
    
    self._previous_streak = streak
    return None
```

### Issue #2: Missing Error Handling for Dependency

- **Severity:** 🔴 Critical (potential crash)
- **Location:** `src/screens/spelling_challenge.py:394`
- **Problem:** The code accesses `self.streak_display.position` without verifying `streak_display` exists and has a valid position property. If streak_display fails to initialize, the game will crash when a 3-word streak is reached.
- **Impact:** Potential game crash when triggering golden boost animation.
- **Recommended Fix:** Add fallback position to prevent crash.

**Current Code:**
```python
if self.streak_display:
    rocket_pos = self.streak_display.position
    self.active_bonus_animation = create_rocket_boost_animation(screen, rocket_pos)
```

**Fixed Code:**
```python
# Use default position as fallback
rocket_pos = (screen.get_width() - 100, screen.get_height() // 2)
if self.streak_display and hasattr(self.streak_display, 'position'):
    rocket_pos = self.streak_display.position
self.active_bonus_animation = create_rocket_boost_animation(screen, rocket_pos)
```

### Issue #3: No Error Handling in Animation Updates

- **Severity:** 🟡 Medium
- **Location:** `src/screens/spelling_challenge.py:427-447`
- **Problem:** If animation updates raise exceptions, the entire game crashes.
- **Recommended Fix:** Add try-catch block around animation updates with fallback to clear animations.

```python
def update_bonus_animation(self, dt: float):
    """Update bonus animation state."""
    try:
        if self.active_bonus_message:
            self.active_bonus_message.update(dt)
            if self.active_bonus_message.is_complete():
                self.active_bonus_message = None
        
        if self.active_bonus_animation:
            self.active_bonus_animation.update(dt)
            if self.active_bonus_animation.is_complete():
                self.active_bonus_animation = None
                if self.streak_bonus_manager:
                    self.streak_bonus_manager.clear_active_bonus()
    except Exception as e:
        logger.error(f"Error updating bonus animation: {e}")
        # Fallback: clear animations to prevent crash
        self.active_bonus_animation = None
        self.active_bonus_message = None
```
