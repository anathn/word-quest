## Overview

This story implements the core word presentation system that displays spelling words to students with both audio pronunciation and visual text, including starter letter hints.

## User Story

As a 3rd grade student, I want to see and hear spelling words with some starter letters so that I can begin spelling the word with guidance.

## Acceptance Criteria

### Functional Requirements

- [ ] Word is spoken aloud via text-to-speech when challenge begins
- [ ] Word is displayed visually on screen in large, clear text
- [ ] First 2-3 letters are shown as starter hints (based on difficulty level)
- [ ] Definition or context sentence is displayed below the word
- [ ] Audio can be replayed on demand via button press
- [ ] Starter letters are visually distinct from letters student must type

### Non-Functional Requirements

- [ ] Performance: Word display appears within 200ms of challenge start
- [ ] Accessibility: All text meets WCAG 2.1 AA contrast ratios (4.5:1 minimum)
- [ ] Error handling: Graceful fallback if TTS engine unavailable (show text only)

### Code Structure Suggestions

```python
# src/screens/spelling_challenge.py
class SpellingChallengeScreen(GameState):
    def __init__(self, word_manager, audio_system):
        self.current_word = None
        self.displayed_letters = []  # Letters student has typed
        self.starter_letters = []    # Pre-displayed hints
        
    def present_word(self, word):
        """Display word with audio and starter hints"""
        self.current_word = word
        self.starter_letters = word.get_starter_letters()
        self.audio_system.speak(word.text)
        
    def render(self, screen):
        self.render_word_display(screen)
        self.render_starter_letters(screen)
        self.render_input_area(screen)

# src/components/audio_system.py
class AudioSystem:
    def __init__(self):
        self.tts_engine = self.init_tts()
        self.audio_available = True
        
    def speak(self, text):
        """Text-to-speech conversion"""
        if self.audio_available:
            self.tts_engine.say(text)
            self.tts_engine.runAndWait()
```
