## Overview

Create engaging letter appearance animations including fade-in effects, bounce animations, and smooth letter-by-letter display during word presentation.

## User Story

As a 3rd grade student, I want letters to appear with fun animations so that spelling feels exciting and dynamic.

## Acceptance Criteria

### Functional Requirements

- [ ] Letters fade in smoothly when displayed (200-300ms transition)
- [ ] Letters have bounce animation on appearance (slight upward bounce)
- [ ] Letters appear in sequence (not all at once) with 100-150ms delay between letters
- [ ] Starter letters have different visual style (may be green tint)
- [ ] Hint-revealed letters have distinct animation (orange highlight)
- [ ] User-typed letters appear instantly with subtle animation
- [ ] Animations are smooth at ≥60 FPS
- [ ] Animation intensity can be reduced for sensitivity preferences

### Non-Functional Requirements

- [ ] Performance: Letter rendering adds <2ms per frame
- [ ] Memory: Animation system uses <5MB RAM
- [ ] Frame rate: Maintain ≥60 FPS during letter animations
- [ ] Accessibility: Animations can be reduced/disabled
- [ ] Scalability: Animations look good at different font sizes

import math

class Easing:
    @staticmethod
    def ease_in_out(t: float) -> float:
        """Smooth ease-in-out interpolation."""
        if t < 0.5:
            return 2 * t * t
        return 1 - math.pow(-2 * t + 2, 2) / 2
    
    @staticmethod
    def ease_out_bounce(t: float) -> float:
        """Bounce effect for letter appearance."""
        n1 = 7.5625
        d1 = 2.75
        if t < 1 / d1:
            return n1 * t * t
        elif t < 2 / d1:
            return n1 * (t -= 1.5 / d1) * t + 0.75
        elif t < 2.5 / d1:
            return n1 * (t -= 2.25 / d1) * t + 0.9375
        return n1 * (t -= 2.625 / d1) * t + 0.984375
    
    @staticmethod
    def ease_out_back(t: float) -> float:
        """Slight overshoot/back effect."""
        c1 = 1.70158
        c3 = c1 + 1
        return 1 + c3 * math.pow(t - 1, 3) + c1 * math.pow(t - 1, 2)

class LetterAnimator:
    ANIMATION_DURATION = 0.3  # 300ms
    LETTER_DELAY = 0.15  # 150ms between letters
    
    def __init__(self, font: pygame.font.Font, base_color: tuple):
        self.font = font
        self.base_color = base_color
        self.letters = []
        self.current_index = 0
        self.is_animating = False
    
    def set_word(self, word: str, starter_count: int = 2):
        """Initialize animator with word and starter letters."""
        self.letters = []
        for i, char in enumerate(word):
            is_starter = i < starter_count
            letter = LetterRenderer(char, self.font, self.base_color)
            if is_starter:
                letter.set_starter_style()
            self.letters.append(letter)
        self.current_index = 0
    
    def start_animation(self):
        """Begin letter-by-letter animation."""
        self.is_animating = True
        self.current_index = 0
        # Start first letter
        if self.letters:
            self.letters[0].start_animation()
    
    def update(self, delta_time: float):
        """Update all letter animations."""
        if not self.is_animating:
            return
        
        # Update all letters
        for letter in self.letters:
            if letter.state != LetterState.COMPLETE:
                letter.update(delta_time)
        
        # Check if current letter is complete, start next
        if (self.current_index < len(self.letters) and
            self.letters[self.current_index].state == LetterState.COMPLETE):
            self.current_index += 1
            if self.current_index < len(self.letters):
                self.letters[self.current_index].start_animation()
            else:
                self.is_animating = False
    
    def render(self, screen: pygame.Surface, start_position: tuple, 
               spacing: int = 20):
        """Render all letters."""
        x = start_position[0]
        for i, letter in enumerate(self.letters):
            letter.render(screen, (x, start_position[1]))
            x += spacing
    
    def is_complete(self) -> bool:
        """Return if all letters have finished animating."""
        return not self.is_animating
    
    def show_all_instantly(self):
        """Skip animation, show all letters immediately."""
        for letter in self.letters:
            letter.state = LetterState.COMPLETE
            letter.progress = 1.0

### Visual Design Notes

- **Letter Size**: 48pt bold for spelling area (as per PRD)
- **Letter Spacing**: 20px between letters
- **Fade Effect**: Alpha from 0 to 255 over 300ms
- **Bounce Effect**: Letter scales to 110% then back to 100%
- **Bounce Timing**: Peak scale at 50% of animation
- **Starter Letters**: Slight green underline or background
- **Hint Letters**: Orange glow or background highlight
- **Background**: Letters on dark semi-transparent panel for contrast

## Notes & Considerations

### Edge Cases
- **Long Words**: Words with 10+ letters may need faster delays
- **Short Words**: 1-2 letter words should still animate
- **Accessibility Off**: When animations disabled, letters appear instantly
- **Performance**: On low-end hardware, reduce animation complexity

### Implementation Tips
- Use surface caching for letter rendering (don't re-render each frame)
- Pre-compute letter positions before animation starts
- Use delta time for frame-rate independent animation
- Consider using sprite groups for animation management

### Performance Optimization
- Cache letter surfaces at standard sizes
- Use integer coordinates for rendering
- Limit bounce/stretch calculations to simple scaling
- Use Pygame's built-in alpha for fade (hardware accelerated)

### Animation Variations
- Could add letter rotation for playful feel
- Could add glow effect on completion
- Could add particle burst on final letter
- Future: Different animation themes to choose

### Accessibility
- Always provide option to reduce or disable animations
- Ensure text remains readable during animations
- Color indications should have non-color backup
- Test with motion sensitivity in mind
