## Overview

This story implements an achievement badge system that unlocks special badges when students reach specific milestones. Badges provide long-term goals and recognition for various types of accomplishments beyond just streaks.

## User Story

As a 3rd grade student, I want to earn badges for achieving different milestones so that I can collect them and feel proud of my diverse accomplishments.

## Acceptance Criteria

### Functional Requirements

- [ ] Define badge types with unlock conditions:
  - **Speed Speller**: Complete 10 words in under 5 minutes
  - **Perseverance**: Master a word after 5+ attempts
  - **Perfect Planet**: Get 5/5 correct on first attempt
  - **Streak Master**: Achieve a 10-word streak
  - **Word Warrior**: Master 25 words total
  - **Comeback Kid**: Correct answer after 3+ incorrect attempts
- [ ] Track unlock conditions in real-time
- [ ] Display "Badge Unlocked!" notification when earned
- [ ] Store unlocked badges in student profile
- [ ] Persist badges across game sessions
- [ ] Show progress toward next badge (e.g., "3/10 words for Speed Speller")

### Non-Functional Requirements

- [ ] Performance: Badge unlock check completes in <50ms
- [ ] Storage: Badge data stored efficiently in JSON
- [ ] Accessibility: Badge notifications use text + icon + color

### UI/UX Requirements

- [ ] Badge notification: Center-screen with animation
- [ ] Badge icon: 64x64px with unique design per badge
- [ ] Progress display: Optional showing in student progress view
- [ ] Notification duration: 3 seconds then auto-dismiss

### Badge Progress Tracking

```python
# Track progress for incremental badges
@dataclass
class BadgeTracker:
    # Speed Speller progress
    speed_speller_words: int = 0
    speed_speller_start_time: Optional[float] = None
    
    # Perseverance tracking
    current_word_attempts: int = 0
    hardest_word_attempts: int = 0
    
    # Perfect Planet tracking
    perfect_planets_earned: int = 0
    
    # Streak Master (uses STORY-004-01 streak tracker)
    max_streak_achieved: int = 0
    
    # Word Warrior
    total_words_mastered: int = 0
    
class BadgeProgressTracker:
    def __init__(self, badge_manager: BadgeManager):
        self.badge_manager = badge_manager
        self.tracker = BadgeTracker()
        
    def on_word_started(self):
        self.tracker.speed_speller_start_time = time.time()
        
    def on_word_completed(self, attempts: int, hints_used: int, is_first_attempt_correct: bool):
        # Update speed speller
        if self.tracker.speed_speller_start_time:
            elapsed = time.time() - self.tracker.speed_speller_start_time
            if elapsed < 300:  # 5 minutes
                self.tracker.speed_speller_words += 1
                if self.tracker.speed_speller_words >= 10:
                    self.badge_manager.unlock_badge("speed_speller")
        
        # Update perseverance
        self.tracker.current_word_attempts = attempts
        if attempts >= 5 and self.badge_manager.check_unlock("perseverance"):
            self.badge_manager.unlock_badge("perseverance")
            
        # Update perfect planet (done at planet level)
        
        # Update word warrior
        if is_first_attempt_correct and hints_used == 0:
            self.tracker.total_words_mastered += 1
            if self.tracker.total_words_mastered >= 25:
                self.badge_manager.unlock_badge("word_warrior")

### Visual Design Notes

#### Badge Unlocked Notification

- **Animation Sequence**:
  1. Badge icon zooms in from center (0 → 1.0 scale in 300ms)
  2. Confetti particles burst outward
  3. "BADGE UNLOCKED!" text fades in above icon
  4. Badge name displays below icon
  5. After 3 seconds, fades out over 300ms
  
- **Colors by Rarity**:
  - Common: Silver border
  - Uncommon: Bronze border
  - Rare: Gold border
  - Legendary: Rainbow animated border

- **Layout**:
  ```
  ┌─────────────────────────────────────┐
