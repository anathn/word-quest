## Overview

This story implements the progressive hint system that provides escalating assistance when students struggle with spelling, from letter count hints to full letter-by-letter revelation.

## User Story

As a 3rd grade student who is stuck, I want helpful hints that gradually give more information so that I can figure out the word without feeling frustrated.

## Acceptance Criteria

### Functional Requirements

- [ ] First hint (after 1 incorrect): Show letter count ("This word has 7 letters")
- [ ] Second hint (after 2 incorrect): Reveal next letter in sequence
- [ ] Third hint (after 3 incorrect): Reveal next letter
- [ ] Continue letter-by-letter until student types answer or all letters revealed
- [ ] No penalty or negative feedback for using hints
- [ ] Hints can be requested via "Need Help?" button
- [ ] All hint usage tracked for progress analytics

### Non-Functional Requirements

- [ ] Performance: Hint appears within 200ms of request
- [ ] Accessibility: Hints use both visual text and audio description
- [ ] Error handling: Handle edge case when word is 1-2 letters (skip to full reveal)

### Code Structure Suggestions

```python
# src/components/hint_manager.py
class HintManager:
    def __init__(self, word: str):
        self.word = word
        self.hint_level = 0  # 0 = no hints, 1 = letter count, 2+ = letter reveals
        self.revealed_letters = set()  # Indices of revealed letters
        
    def get_next_hint(self) -> str:
        """Generate next hint based on current level"""
        if self.hint_level == 0:
            return f"This word has {len(self.word)} letters"
        elif self.hint_level < len(self.word):
            idx = self.hint_level
            letter = self.word[idx]
            self.revealed_letters.add(idx)
            return f"The {idx+1}th letter is '{letter}'"
        else:
            return f"The word is: {self.word}"
    
    def escalate(self):
        """Move to next hint level"""
        self.hint_level += 1
    
    def get_revealed_letters(self) -> list:
        """Return word with revealed letters shown, others as underscores"""
        result = []
        for i, letter in enumerate(self.word):
            if i in self.revealed_letters:
                result.append(letter)
            else:
                result.append('_')
        return ' '.join(result)
```
