## Overview

Review and validate the game's color palette to ensure it is color-blind safe, avoiding red-green combinations that are indistinguishable to users with deuteranopia (green-blind) and protanopia (red-blind). This story ensures all visual elements are accessible to the ~8% of males and ~0.5% of females with color vision deficiencies.

## User Story

As a student with color blindness, I want the game colors to be distinguishable so that I can play without confusion or frustration.

## Acceptance Criteria

### Functional Requirements

- [ ] All UI colors validated against color-blind simulators
- [ ] No critical information conveyed by color alone
- [ ] Red-green combinations replaced with distinguishable alternatives
- [ ] Feedback colors (correct/incorrect) use shape + color
- [ ] Planet colors are distinguishable for all types of color blindness
- [ ] Text contrast ratios meet WCAG 2.1 AA (4.5:1 minimum)
- [ ] Theme configuration updated with color-blind safe palette

### Non-Functional Requirements

- [ ] Performance: Color changes have zero performance impact
- [ ] Accessibility: Tested with at least 3 color-blind simulation tools
- [ ] Error handling: Graceful fallback if invalid color specified

### UI/UX Requirements

- [ ] Correct answer feedback: Green + checkmark shape
- [ ] Incorrect answer feedback: Blue + X shape (NOT red)
- [ ] Streak indicators: Yellow/gold flame (visible to all)
- [ ] Planet colors: Blue, orange, purple, brown, gold (not red/green/teal)

### Visual Design Notes

#### Color-Blind Safe Palette

| Element | Color | Hex | Rationale |
|---------|-------|-----|-----------|
| Background | Deep Blue | #1a1a3e | Good contrast for all types |
| Primary Text | White | #ffffff | Maximum contrast |
| Correct Feedback | Green | #4CAF50 | OK with shape indicator |
| Incorrect Feedback | **Blue** | #2196F3 | NOT red - distinguishable |
| Streak | Gold/Yellow | #FFD700 | Visible to all types |
| Highlight | Orange | #FF9800 | Distinguishable from blue/green |
| Planet 1 | Orange | #FF9800 | Distinguishable |
| Planet 2 | Blue | #2196F3 | Distinguishable |
| Planet 3 | Purple | #9C27B0 | Distinguishable (blue+red) |
| Planet 4 | Brown | #795548 | Instead of green |
| Planet 5 | **Gold** | **#C9AA50** | Instead of red - **chosen over teal** (Blue vs Teal not distinguishable to deuteranopic users)

#### Color-Blind Simulation Results

| Color | Normal Vision | Deuteranopia | Protanopia | Tritanopia |
|-------|--------------|---------------|-------------|-------------|
| #1a1a3e | Deep Blue | Deep Blue | Deep Blue | Deep Blue |
| #4CAF50 (former green) | Green | Brownish | Grayish-Brown | Blue-Green |
| #2196F3 (new incorrect) | Blue | Blue | Blue | Yellow-ish |
| #FF9800 | Orange | Orange | Orange | Yellow |
| #9C27B0 | Purple | Purple | Purple | Blue |
| #00BCD4 (TEAL - NOT USED) | Teal | Teal | Blue | **NOT DISTINGUISHABLE from blue** |
| **#C9AA50 (GOLD - ACTUAL)** | Gold | **Brownish-Gold** | **Yellow** | **Distinguishable from blue** |

**Key Changes from Original Palette:**
- ❌ **Removed**: Red (#F44336) - indistinguishable from green
- ❌ **Removed**: Pure Green (#00FF00) - problematic for red-blind
- ✅ **Added**: Blue (#2196F3) for incorrect feedback  
- ✅ **Added**: Brown (#795548) instead of green planet  
- ❌ **Teal rejected**: Blue vs Teal = 17.3 distance (below 80 threshold for deuteranopia)  
- ✅ **Gold chosen**: Blue vs Gold = 201.9 distance (distinguishable for all types)

## Notes & Considerations

### Edge Cases

- **Mixed Vision Users**: Some users have partial color blindness - test partially desaturated colors
- **Lighting Conditions**: Colors viewed in bright vs dark rooms - ensure contrast holds up
- **Monitor Variations**: Different displays render colors differently - test on multiple screens

### Implementation Tips

1. **Use a Validator Early**: Run color validation as part of development, not just at the end
2. **Document Rationale**: Explain WHY certain colors were chosen for future reference
3. **Shape First Design**: Design for shape/color separation from the start
4. **Test Tools**: 
   - Coblis (web-based simulator)
   - Color Oracle (desktop app)
   - Chrome DevTools (emulation)
   - fovea (CLI tool)
