## Overview

This story implements the words mastered counter that displays cumulative progress as a simple "X/Y words" format, updating in real-time during gameplay.

## User Story

As a 3rd grade student, I want to see how many words I've mastered so far so that I can see my overall progress.

## Acceptance Criteria

### Functional Requirements

- [ ] Track mastered words list (words with 100% first-attempt accuracy)
- [ ] Display as "X/Y words" format (e.g., "23/50 words")
- [ ] Update in real-time when word is mastered
- [ ] Persist mastered words list to progress data
- [ ] Load mastered words on profile selection

### Non-Functional Requirements

- [ ] Performance: Counter updates within 100ms of word completion
- [ ] Accessibility: Counter text meets WCAG contrast ratios
- [ ] Error handling: Handle empty mastered list gracefully ("0/50 words")

### Code Structure Suggestions

```python
# src/components/progress_tracker.py
class ProgressTracker:
    def __init__(self, student_id: str, data_store: DataStore):
        self.student_id = student_id
        self.data_store = data_store
        self.progress_data = data_store.load(student_id)
        self.mastered_words = set(self.progress_data.get('mastered_words', []))
        
    def mark_word_mastered(self, word: str):
        """Add word to mastered list if not already mastered"""
        if word not in self.mastered_words:
            self.mastered_words.add(word)
            self.progress_data['mastered_words'] = list(self.mastered_words)
            self.data_store.save(self.student_id, self.progress_data)
            
    def get_mastered_count(self) -> int:
        return len(self.mastered_words)
    
    def get_total_words(self) -> int:
        """Get total words in all word lists"""
        # This might come from word manager
        return self.word_manager.get_total_word_count()
    
    def get_progress_text(self) -> str:
        mastered = self.get_mastered_count()
        total = self.get_total_words()
        return f"{mastered}/{total} words"

# src/ui/progress_display.py
class ProgressDisplay:
    def __init__(self, progress_tracker: ProgressTracker):
        self.tracker = progress_tracker
        self.font = pygame.font.Font(None, 36)
        
    def render(self, screen, position: tuple):
        text = self.tracker.get_progress_text()
        text_surface = self.font.render(text, True, WHITE)
        screen.blit(text_surface, position)
```
