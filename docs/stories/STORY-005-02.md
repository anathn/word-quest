## Overview

Create an animated rocket ship character that serves as the player's avatar, featuring engine flame animations, movement animations, and color customization options.

## User Story

As a 3rd grade student, I want to pilot my own customizable rocket through space so that I feel connected to my space exploration journey.

## Acceptance Criteria

### Functional Requirements

- [ ] Rocket sprite displays with distinct nose, body, wings, and engine
- [ ] Engine flames animate with flickering/fading effect
- [ ] Rocket responds to user input with smooth movement animations
- [ ] Rocket displays in main menu, gameplay, and transition screens
- [ ] Color customization applies to rocket body (8 preset colors)
- [ ] Rocket animation frames cycle smoothly (≥30 FPS)
- [ ] Rocket maintains consistent size across all screens
- [ ] Rocket rotation/tilt during movement for visual interest

### Non-Functional Requirements

- [ ] Performance: Rocket rendering adds <3ms per frame
- [ ] Memory: Rocket sprite assets use <10MB RAM
- [ ] Animation: Maintain smooth ≥60 FPS for engine flames
- [ ] Scalability: Rocket scales proportionally at different sizes
- [ ] Persistence: Color choice saved to player profile

4. **RocketConfig** - Configuration manager for rocket colors. Handles loading/saving player preferences.

## Notes & Considerations

### Edge Cases
- **First Launch**: Default to white rocket if no preference saved
- **Invalid Color**: Fall back to default if saved color is invalid
- **Window Resize**: Rocket scales to maintain visibility
- **Low FPS**: Reduce particle count if performance drops

### Implementation Tips
- Build rocket from geometric shapes (circles, triangles) for procedural generation
- Use surface masking for smooth edge rendering
- Cache rocket at different sizes to avoid repeated scaling
- Use additive blending for flame effect (glowing appearance)

### Performance Optimization
- Pre-render rocket at common sizes (64, 96, 128)
- Limit max flame particles to 20 for MVP
- Reuse flame particle objects (object pooling)
- Use integer coordinates for rendering when possible

### Accessibility
- Ensure rocket color is visible against space background
- Provide high-contrast outline option
- Animation intensity can be reduced for sensitivity
