## Overview

This story implements a visible streak counter that tracks consecutive correct answers and displays the current streak to motivate students during gameplay. The streak resets on incorrect answers but does not penalize the student.

## User Story

As a 3rd grade student, I want to see how many words I've spelled correctly in a row so that I feel motivated to keep going and build a longer streak.

## Acceptance Criteria

- [ ] Streak resets to 0 on incorrect answer

```python
from dataclasses import dataclass
from typing import Optional

@dataclass
class StreakData:
    current_streak: int = 0
    best_streak: int = 0
    streak_start_time: Optional[float] = None
    
class StreakTracker:
    def __init__(self):
        self.current_streak = 0
        self.best_streak = 0
        
    def record_correct_answer(self) -> int:
        """Increment streak and return new streak value"""
        self.current_streak += 1
        if self.current_streak > self.best_streak:
            self.best_streak = self.current_streak
        return self.current_streak
    
    def record_incorrect_answer(self) -> None:
        """Reset streak to 0 (no penalty, just reset)"""
        self.current_streak = 0
    
    def get_current_streak(self) -> int:
        return self.current_streak
    
    def get_best_streak(self) -> int:
        return self.best_streak
    
    def reset_session(self) -> None:
        """Start new session (streak starts at 0, best persists)"""
        self.current_streak = 0
```

## UI/UX Requirements

### Screens Involved

- **Spelling Challenge Screen**: Streak display in top-right corner

### User Interactions

1. First correct answer → Streak display appears with "1"
2. Each subsequent correct answer → Number increments, small pulse animation
3. Incorrect answer → Streak resets to 0, display disappears

### Visual Design Notes

- **Streak Display Location**: Top-right corner (800, 20) position
- **Flame Icon**: Golden/orange flame (24x24px or 32x32px)
- **Number Font**: 48pt bold, golden yellow (#FFD700)
- **Background**: Semi-transparent dark circle behind number for contrast
- **Animation**: 
  - When streak increments: 100ms scale up/down pulse (1.0 → 1.2 → 1.0)
  - When streak resets: Fade out over 200ms, then hide

- **No Streak State**: Display hidden entirely (not showing "0")
- **High Streak visual**: At streak 5+, add star particles around flame

### Implementation Details

**StreakTracker Component:**
- Tracks `current_streak` and `best_streak`
- `record_correct_answer()`: Increments streak, updates best if needed
- `record_incorrect_answer()`: Resets current streak to 0 (non-punitive)
- `reset_session()`: Resets current streak, preserves best streak
- `reset()`: Full reset for testing (clears both current and best)
- Serialization: `to_dict()` / `from_dict()` for persistence

**StreakDisplay Component:**
- Position: Top-right corner (800, 20)
- Flame icon: Procedurally rendered 32x32 golden/orange gradient
- Number font: 48pt, golden yellow (#FFD700)
- Background: Semi-transparent dark circle for contrast
- Pulse animation: 100ms scale animation (1.0 → 1.2 → 1.0) on increment
- Hidden state: Display completely hidden when streak is 0 (non-punitive design)
- Star particles: 5 animated stars for streak 5+ (special effect)
- Performance: <5ms render time

**Integration Points:**
- ProgressTracker.record_correct_answer() → calls StreakTracker.record_correct_answer()
- ProgressTracker.record_incorrect_answer() → calls StreakTracker.record_incorrect_answer()
- SpellingChallengeScreen._on_auto_advance() → updates streak display on correct
- SpellingChallengeScreen._on_feedback_shown() → resets streak on RETRY feedback
- SessionTracker tracks _session_best_streak for session summaries
