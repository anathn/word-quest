## Overview

This story implements a space-themed sticker collection system that rewards students with collectible stickers for achieving milestones. Stickers can be viewed in a personal collection, encouraging continued play and exploration.

## User Story

As a 3rd grade student, I want to collect space stickers for my achievements so that I can display them proudly and work toward completing my collection.

## Acceptance Criteria

### Functional Requirements

- [ ] Define sticker types with unlock conditions:
  - **Planet Winner**: Complete any planet (5/5 words)
  - **Perfect Score**: Complete a planet with 5/5 first-attempt correct
  - **Speed Demon**: Complete a planet in under 3 minutes
  - **Hint Helper**: Successfully solve a word after using all hints
  - **Comeback Champion**: Complete a planet after failing it once before
  - **Galaxy Explorer**: Complete all planets in a solar system (3 planets)
  - **Streak Queen/King**: Achieve 15-word streak
  - **Word Master**: Master 50 words total
  - **Early Bird**: Play game before 9am
  - **Night Owl**: Play game after 7pm
- [ ] Each sticker has unique artwork and name
- [ ] Unlock notifications display when sticker earned
- [ ] Sticker collection view shows all earned stickers
- [ ] Locked stickers show as grayed out with lock icon
- [ ] Collection persists across game sessions
- [ ] Clicking sticker shows unlock date and conditions

- [ ] Accessibility: Sticker names read by TTS, descriptions available

### UI/UX Requirements

- [ ] Sticker size: 80x80px in collection

### Sticker Event Tracking

```python
# src/components/sticker_tracker.py
# Tracks complex unlock conditions that require ongoing monitoring

class StickerProgressTracker:
    def __init__(self, sticker_manager: StickerManager):
        self.manager = sticker_manager
        self.tracker = {
            "solar_systems_completed": set(),  # Track which solar systems
            "failed_then_cleared_planets": set(),  # Planets failed before clearing
            "current_streak": 0,
            "words_mastered": 0,
            "games_before_9am": 0,
            "games_after_7pm": 0,
        }
        
    def on_planet_failed(self, planet_id: str):
        """Track planet failure for Comeback Champion"""
        self.tracker["failed_then_cleared_planets"].add(planet_id)
        
    def on_planet_cleared(self, planet_id: str, solar_system: str):
        """Track planet clearance"""
        if planet_id in self.tracker["failed_then_cleared_planets"]:
            # Unlock Comeback Champion
            self.manager._unlock("comeback_champion", {"planet": planet_id})
            
        self.tracker["solar_systems_completed"].add(solar_system)
        if len(self.tracker["solar_systems_completed"]) >= 3:
            self.manager._unlock("galaxy_explorer")
            
    def on_streak_update(self, streak: int):
        """Track streak for Streak Queen/King"""
        if streak >= 15:
            self.manager._unlock("streak_queen", {"streak": streak})
            
    def on_word_mastered(self):
        """Track mastered words"""
        self.tracker["words_mastered"] += 1
        if self.tracker["words_mastered"] >= 50:
            self.manager._unlock("word_master")
            
    def on_game_started(self, hour: int):
        """Track time of day for Early Bird/Night Owl"""
        if hour < 9:
            self.tracker["games_before_9am"] += 1
            if self.tracker["games_before_9am"] >= 1:
                self.manager._unlock("early_bird")
        elif hour >= 19:
            self.tracker["games_after_7pm"] += 1
            if self.tracker["games_after_7pm"] >= 1:
                self.manager._unlock("night_owl")
```

### Integration Points

The StickerManager integrates with ProgressTracker through the following hooks:

1. **Session Start**: `start_session(hour)` - Checks time for Early Bird/Night Owl
2. **Planet Start**: `start_planet()` - Records planet start time for Speed Demon
3. **Planet Complete**: `on_planet_completed()` - Checks multiple sticker conditions
4. **Planet Fail**: `on_planet_failed()` - Marks planet for Comeback Champion tracking
5. **Full Hint**: `on_full_hint_solved()` - Called from hint system when all hints used
6. **Streak Update**: `on_streak_update(streak)` - Called after each correct answer
7. **Word Mastered**: `on_word_mastered()` - Called when word reaches mastery
8. **Session End**: `end_session()` - Persists unlocked stickers to student profile

### Known Limitations / Future Enhancements

1. **Sticker Artwork**: Current implementation uses procedurally generated icons. Actual PNG artwork can be added later for better visual quality.

2. **Galaxy Explorer**: The solar system completion tracking is simplified. Full implementation would require tracking which planetary systems have been completed.

3. **Streak King/Queen**: Gender-based sticker assignment uses `is_female` flag. Could be made more flexible with customizable pronouns/names.

4. **Performance**: For MVP, all stickers are rendered at once. For very large collections (100+ stickers), lazy-loading and virtual scrolling would improve performance.

5. **Audio**: Unlock sounds not implemented (deferred to STORY-005-03 SFX story).

6. **Expansion**: The system is designed to be easily extended:
   - Add new stickers by updating `data/sticker_definitions.json`
   - Add new categories by updating the filter buttons
   - Add new unlock conditions by implementing event handlers
