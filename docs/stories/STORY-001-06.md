## Overview

This story implements the visual progression system that animates the rocket traveling between planets and updates the player's position in the game world.

## User Story

As a 3rd grade student, I want to see my rocket travel to new planets as I complete them so that I can see my progress through space.

## Acceptance Criteria

### Functional Requirements

- [ ] Animation showing rocket traveling to next planet
- [ ] Planet name/number displayed during transition
- [ ] Progress bar updates to show galaxy progress
- [ ] Unlock visual indicator for newly accessible planets
- [ ] Sound effect during transition animation
- [ ] Skip transition animation option for faster gameplay

### Non-Functional Requirements

- [ ] Performance: Animation runs at ≥30 FPS
- [ ] Accessibility: Transition can be skipped or shortened
- [ ] Error handling: Handle case where no next planet available (galaxy complete)

```python
# src/screens/planet_transition.py
class PlanetTransitionScreen(GameState):
    def __init__(self, from_planet: str, to_planet: str, progress_percent: float):
        self.from_planet = from_planet
        self.to_planet = to_planet
        self.progress = progress_percent
        self.animation_progress = 0.0  # 0.0 to 1.0
        self.skip_requested = False
        
    def handle_events(self):
        for event in pygame.event.get():
            if event.type == pygame.KEYDOWN:
                if event.key == pygame.K_SPACE:
                    self.skip_requested = True
                    
    def update(self, delta_time):
        if not self.skip_requested:
            self.animation_progress += delta_time * 0.5  # 2 second animation
        self.animation_progress = min(1.0, self.animation_progress)
        
    def render(self, screen):
        self.render_background(screen)
        self.render_rocket(screen, self.animation_progress)
        self.render_planet_info(screen)
        self.render_progress_bar(screen, self.progress)
```

## UI/UX Requirements

### Screens Involved

- **Planet Transition Screen**: Full-screen animation between planets
- **Main Menu**: Shows planet map with progression

### User Interactions

1. Planet completed → Transition screen appears automatically
2. Animation plays: Rocket flies from current planet to next
3. Planet name displayed: "Traveling to Planet Venus..."
4. Progress bar shows overall galaxy progress
5. Press Space or click to skip animation
6. Animation completes → Next planet challenge begins

### Visual Design Notes

- **Transition Screen**:
  - Background: Star field with motion blur effect
  - Rocket: Animated sprite with engine flames
  - Planets: Show departure and destination planets on screen edges
  - Duration: 2 seconds (can be skipped)
  
- **Planet Map** (Main Menu):
  - Solar system layout with connected planets
  - Completed planets: Green with checkmark
  - Current planet: Pulsing yellow border
  - Locked planets: Grayed out with lock icon
  - Progress bar: Shows galaxy completion percentage

- **Text**:
  - Planet name: 48pt, white, centered
  - Progress: "3/12 planets completed" in 28pt

- **Audio**:
  - Rocket engine hum during travel
  - "Whoosh" sound at animation start
  - Gentle chime at arrival
