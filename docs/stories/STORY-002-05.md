## Overview

This story implements the words needing practice list that identifies words with multiple attempts and displays them sorted by difficulty (most attempts first).

## User Story

As a parent and student, I want to see which words need more practice so that we can focus on challenging words.

## Acceptance Criteria

### Functional Requirements

- [ ] Track words with multiple attempts (not mastered on first try)
- [ ] Display list with attempt counts (e.g., "because - 3 attempts")
- [ ] Sort by attempts (most attempts first = most challenging)
- [ ] Update list after each session
- [ ] Remove words from list once mastered
- [ ] Show top 10 words needing practice (or all if fewer)

### Non-Functional Requirements

- [ ] Performance: List generation completes in < 100ms
- [ ] Accessibility: List items meet contrast requirements
- [ ] Error handling: Handle empty list gracefully ("Great job! No words need practice")

```python
# src/components/progress_tracker.py
class ProgressTracker:
    # ... existing code ...
    
    def get_words_needing_practice(self, limit: int = 10) -> list:
        """Get words that need more practice, sorted by attempts"""
        # Collect all word attempts
        word_attempts = {}  # word -> total attempts
        
        for session in self.progress_data.get('sessions', []):
            for word_data in session.get('words', []):
                word = word_data['word']
                attempts = word_data.get('attempts', 1)
                
                # Only count if not mastered (not first-attempt correct)
                if not word_data.get('first_attempt_correct', False):
                    word_attempts[word] = word_attempts.get(word, 0) + attempts
        
        # Exclude already mastered words
        mastered = set(self.progress_data.get('mastered_words', []))
        word_attempts = {w: a for w, a in word_attempts.items() if w not in mastered}
        
        # Sort by attempts (descending) and return top N
        sorted_words = sorted(word_attempts.items(), key=lambda x: x[1], reverse=True)
        return sorted_words[:limit]
    
    def format_practice_list(self) -> list:
        """Format practice list for display"""
        words = self.get_words_needing_practice()
        return [
            {'word': word, 'attempts': attempts, 'label': f"{word} - {attempts} attempts"}
            for word, attempts in words
        ]
```
