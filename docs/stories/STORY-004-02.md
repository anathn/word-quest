## Overview

This story implements special visual celebration animations that trigger at streak milestones (3-word and 5-word streaks). These bonus animations provide extra motivation and visual reward for maintaining consecutive correct answers.

## User Story

As a 3rd grade student, I want to see special celebrations when I build a good streak so that I feel extra proud and motivated to keep going.

- [ ] 5-word streak triggers special planet discovery animation

- [ ] Each milestone animation plays only once per session
- [ ] Sound effects play with animations (see STORY-005-03 for SFX)

```python
# src/components/streak_bonus.py
from dataclasses import dataclass
from typing import Callable, List
from enum import Enum

class BonusType(Enum):
    GOLDEN_ROCKET = "golden_rocket"
    PLANET_DISCOVERY = "planet_discovery"

@dataclass
class BonusMilestone:
    streak_threshold: int
    bonus_type: BonusType
    message: str
    triggered: bool = False
    
class StreakBonusManager:
    def __init__(self):
        self.milestones = [
            BonusMilestone(3, BonusType.GOLDEN_ROCKET, "3-word streak! Golden Boost!"),
            BonusMilestone(5, BonusType.PLANET_DISCOVERY, "5-word streak! New Planet Discovered!"),
        ]
        self.active_bonus: Optional[BonusMilestone] = None
        
    def check_milestone(self, streak: int) -> Optional[BonusMilestone]:
        """Check if streak triggered a bonus milestone"""
        for milestone in self.milestones:
            if streak == milestone.streak_threshold and not milestone.triggered:
                milestone.triggered = True
                return milestone
        return None
    
    def reset_session(self):
        """Reset all milestones for new session"""
        for milestone in self.milestones:
            milestone.triggered = False
        self.active_bonus = None
```

## UI/UX Requirements

### Screens Involved

- **Spelling Challenge Screen**: Overlay animations on top of game

### User Interactions

1. Student reaches 3-word streak → Golden rocket boost plays
2. Student reaches 5-word streak → Planet discovery animation plays
3. Animations auto-dismiss after 2-3 seconds

### Visual Design Notes

#### Golden Rocket Boost (3-word streak)

- **Trigger**: When current streak reaches 3
- **Visual Elements**:
  - Rocket engine flames turn from orange to GOLD (#FFD700)
  - Golden particle trail follows rocket (10-15 particles)
  - Subtle glow effect around rocket (white/yellow gradient)
  - Screen shake: tiny 2-pixel shake for impact
- **Message**: "🔥 3-word streak! GOLDEN BOOST! 🚀"
  - Font: 48pt bold, golden yellow
  - Position: Center screen, fades in/out
  - Duration: 2 seconds display
- **Animation Duration**: 1.5 seconds
- **Sound**: Layered celebratory chime (ascending C-E-G with octave)

#### Planet Discovery (5-word streak)

- **Trigger**: When current streak reaches 5
- **Visual Elements**:
  - Camera pans briefly to show NEW planet appearing in distance
  - Planet has unique color/pattern (not seen before)
  - Sparkle effect: white star particles exploding outward
  - Planet "blooms" from small dot to full size
  - Rocket flies toward planet briefly then returns
- **Message**: "✨ 5-word streak! NEW PLANET DISCOVERED! ✨"
  - Font: 56pt bold, white with gold outline
  - Position: Center screen
  - Duration: 2.5 seconds display
- **Animation Duration**: 2.5 seconds
- **Sound**: Victory fanfare (ascending arpeggio)

### Animation Timing

```
Word Correct → Streak becomes 3
   ↓ (100ms)
Bonus check detects milestone
   ↓ (50ms)
Message fades in (200ms)
   ↓
Rocket boost animation plays (1500ms)
   ↓
Message fades out (300ms)
   ↓
Back to normal gameplay
```

### Positioning

**Golden Rocket Boost**:
- Overlay on current rocket position
- Message at center, slightly above rocket
- Particles trail behind rocket

**Planet Discovery**:
- Brief camera offset to show "discovery area" (right side of screen)
- New planet emerges from edge, moves to center
- Rocket flies toward planet, does loop, returns to position
- Sparkles surround planet
