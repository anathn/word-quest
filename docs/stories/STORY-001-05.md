## Overview

This story implements the planet completion logic that tracks word performance within a planet (5-word set) and determines mastery based on the 80% threshold.

## User Story

As a 3rd grade student, I want to know when I've completed a planet and what happens next so that I understand my progress and what to do.

## Acceptance Criteria

### Functional Requirements

- [x] Each planet contains exactly 5 words
- [x] 4/5 correct (80%) unlocks next planet
- [x] 2-3/5 correct replays same planet with same words
- [x] 0-1/5 correct triggers parent notification flag
- [x] Track first-attempt accuracy for each word
- [x] Display progress within planet (e.g., "3/5 words mastered")
- [x] Planet completion screen shows results before proceeding

### Non-Functional Requirements

- [x] Performance: Results calculated and displayed within 500ms of final word
- [x] Accessibility: Results communicated both visually and audio
- [x] Error handling: Handle crash/restart mid-planet (resume from last saved word)

### Code Structure Suggestions

```python
# src/components/planet_manager.py
class PlanetManager:
    def __init__(self, planet_id: str, word_list: list):
        self.planet_id = planet_id
        self.words = word_list[:5]  # 5 words per planet
        self.results = []  # List of word results
        
    def record_word_result(self, word: str, correct: bool, attempts: int):
        """Record result for a word"""
        self.results.append({
            'word': word,
            'correct': correct,
            'attempts': attempts,
            'first_attempt': attempts == 1
        })
    
    def get_completion_status(self) -> dict:
        """Determine planet completion status"""
        total = len(self.results)
        if total < 5:
            return {'status': 'in_progress', 'progress': f"{total}/5"}
        
        first_attempt_correct = sum(1 for r in self.results if r['first_attempt'])
        
        if first_attempt_correct >= 4:
            return {'status': 'completed', 'unlock_next': True}
        elif first_attempt_correct >= 2:
            return {'status': 'retry', 'unlock_next': False}
        else:
            return {'status': 'needs_help', 'unlock_next': False, 'notify_parent': True}
```
