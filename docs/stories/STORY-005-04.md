## Overview

Implement ambient space background music that creates an engaging atmosphere while remaining non-distracting during gameplay, with full volume control and toggle functionality.

## User Story

As a 3rd grade student, I want to hear ambient space music while playing so that the game feels immersive and enjoyable.

## Acceptance Criteria

### Functional Requirements

- [ ] Looping ambient space track plays during gameplay
- [ ] Music is low volume by default (30% or lower)
- [ ] On/off toggle available in settings
- [ ] Smooth fade in/out on state changes (500ms transition)
- [ ] Different music tracks for different game states:
  - Main menu: Welcoming, exploratory
  - Gameplay: Calm, focused ambient
  - Celebration: Uplifting, triumphant
  - Parent dashboard: Neutral, professional
- [ ] Music volume adjustable (0-100%)
- [ ] Music state persists across game sessions
- [ ] Music playback works on Windows, macOS, Linux

### Non-Functional Requirements

- [ ] Performance: Music playback adds <2ms overhead
- [ ] Memory: Audio assets use <30MB RAM
- [ ] Format: Support OGG format for efficient looping
- [ ] Fade transitions complete in 500ms without audio artifacts
- [ ] Cross-platform: Music works on all target platforms

### User Interactions

1. Game launches → Main menu music fades in
2. Student starts game → Music transitions to gameplay track
3. Student completes word → Music continues (no interruption)
4. Student reaches celebration → Brief triumphant track plays
5. Student opens settings → Can adjust volume or mute music

### Audio Design Notes

- **Main Menu Music**: 
  - Tempo: 60-70 BPM
  - Style: Welcoming, exploratory, inviting
  - Instruments: Soft synth pads, gentle melody
  - Duration: 2-3 minute loop
  
- **Gameplay Music**:
  - Tempo: 50-60 BPM
  - Style: Calm, ambient, non-distracting
  - Instruments: Low-volume synth pads, minimal melody
  - Duration: 3-5 minute loop (extended for focus)
  - IMPORTANT: Should not compete with SFX
  
- **Celebration Music**:
  - Tempo: 80-90 BPM
  - Style: Uplifting, triumphant, joyful
  - Instruments: Bright synths, major key
  - Duration: 15-30 seconds (one-time play)
  
- **Dashboard Music**:
  - Tempo: 60 BPM
  - Style: Neutral, professional
  - Instruments: Subtle ambient
  - Duration: 2 minute loop

## Notes & Considerations

### Edge Cases
- **No Audio Hardware**: Gracefully handle missing audio drivers
- **Music File Missing**: Fall back to procedural generation or silence
- **Rapid State Changes**: Queue transitions to prevent skipping
- **First Launch**: Default music on, volume at 30%

### Implementation Tips
- Use OGG format for efficient looping and small file size
- Store music in prepYGAME's music channel (not sound channel)
- Use fadeout/fadein for smooth transitions
- Pre-load tracks before playing to avoid lag

### Procedural vs External Audio

**For MVP (Procedural)**:
- Generate ambient pads using sine waves
- Add reverb effect for space feel
- Create simple arpeggios for melody
- Store as generated waveforms (no external files)

**For Release (External)**:
- Commission original ambient music
- Use professional audio engineer
- Create variations to prevent repetition
- Normalize volume across tracks

### Performance Optimization
- Pre-load all music tracks at startup
- Use Pygame's music channel (separate from SFX)
- Stream from disk for longer tracks if needed
- Cache generated music if using procedural

### Accessibility
- Ensure music is non-distracting at default volume
- Provide clear mute option
- Music should enhance, not compete with gameplay
- Consider sound sensitivity preferences
