## Overview

This story implements a visual line graph showing attempts per word over time (weeks), demonstrating fluency improvement as the student practices more.

## User Story

As a parent, I want to see a visual graph of my child's progress over time so that I can understand learning trends and improvement.

## Acceptance Criteria

### Functional Requirements

- [ ] Line graph showing average attempts per word over weeks
- [ ] X-axis: weeks (last 8 weeks or all data if less)
- [ ] Y-axis: average attempts (1.0 to 3.0 scale)
- [ ] Trend line showing improvement (downward slope = better)
- [ ] Data points for each week
- [ ] Hover/click shows exact values
- [ ] Export graph as image for sharing

### Non-Functional Requirements

- [ ] Performance: Graph renders in < 500ms
- [ ] Accessibility: Graph has text summary alternative
- [ ] Error handling: Handle insufficient data gracefully ("Need more sessions")

class GraphRenderer:
    def __init__(self, width: int = 600, height: int = 300):
        self.width = width
        self.height = height
        self.margin = {'top': 40, 'right': 40, 'bottom': 60, 'left': 60}
        
    def render(self, screen, data: list):
        """Render line graph"""
        # Calculate plot area
        plot_left = self.margin['left']
        plot_right = self.width - self.margin['right']
        plot_top = self.margin['top']
        plot_bottom = self.height - self.margin['bottom']
        
        # Draw axes
        self.draw_axes(screen, plot_left, plot_right, plot_top, plot_bottom)
        
        # Draw trend line
        self.draw_trend_line(screen, data, plot_left, plot_right, plot_top, plot_bottom)
        
        # Draw data points
        self.draw_data_points(screen, data, plot_left, plot_right, plot_top, plot_bottom)
        
        # Draw labels
        self.draw_labels(screen, data, plot_left, plot_right, plot_top, plot_bottom)
    
    def draw_trend_line(self, screen, data, left, right, top, bottom):
        """Draw the main trend line"""
        if len(data) < 2:
            return
            
        points = []
        max_avg = max(d['average'] for d in data) if data else 1
        plot_width = right - left
        plot_height = bottom - top
        
        for i, point in enumerate(data):
            x = left + (i / (len(data) - 1)) * plot_width if len(data) > 1 else left
            y = bottom - (point['average'] / max_avg) * plot_height
            points.append((x, y))
        
        # Draw line connecting points
        pygame.draw.lines(screen, GREEN, False, points, 3)
