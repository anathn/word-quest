## Overview

Implement a high contrast mode that dramatically increases color contrast throughout the game, making text and UI elements more visible for students with low vision, visual impairments, or those viewing in challenging lighting conditions. This mode replaces the default color scheme with high-contrast alternatives.

## User Story

As a student with low vision or who plays in bright/dim lighting, I want a high contrast mode so that text and buttons are clearly visible and easier to read.

# WCAG contrast validation
def validate_contrast(text_color: str, background_color: str) -> bool:
    """Validate that text/background pair meets WCAG AAA (7:1)"""
    def get_luminance(hex_color: str) -> float:
        """Calculate relative luminance"""
        r, g, b = int(hex_color[1:3], 16), int(hex_color[3:5], 16), int(hex_color[5:7], 16)
        
        # sRGB to linear
        def to_linear(c):
            c = c / 255.0
            return c / 12.92 if c <= 0.03928 else ((c + 0.055) / 1.055) ** 2.4
        
        r_lin, g_lin, b_lin = to_linear(r), to_linear(g), to_linear(b)
        return 0.2126 * r_lin + 0.7152 * g_lin + 0.0722 * b_lin
    
    lum1 = get_luminance(text_color)
    lum2 = get_luminance(background_color)
    
    lighter = max(lum1, lum2)
    darker = min(lum1, lum2)
    
    contrast_ratio = (lighter + 0.05) / (darker + 0.05)
    return contrast_ratio >= 7.0  # AAA requirement

    def _load_high_contrast_palette(self) -> Dict[str, tuple]:
        """Load high contrast theme colors"""
        from src.ui.high_contrast_theme import HIGH_CONTRAST_COLORS
        
        palette = {}
        for name, hex_color in HIGH_CONTRAST_COLORS.items():
            # Convert hex to RGB tuple
            r = int(hex_color[1:3], 16)
            g = int(hex_color[3:5], 16)
            b = int(hex_color[5:7], 16)
            palette[name] = (r, g, b)
            
        return palette
        
    def get_color(self, name: str) -> tuple:
        """Get color from current theme"""
        return self.color_palettes[self.current_theme].get(
            name, (255, 255, 255)  # Fallback to white
        )
        
    def set_theme(self, theme_name: str) -> bool:
        """Switch to specified theme"""
        if theme_name in self.color_palettes:
            self.current_theme = theme_name
            self._apply_theme()
            return True
        return False

        """Enable high contrast mode"""

    def disable_high_contrast(self) -> None:
        """Disable high contrast mode (return to default)"""

from dataclasses import dataclass

@dataclass
class AccessibilitySettings:
    """Combined accessibility feature settings"""
    # Caption settings
    captions_enabled: bool = True
    caption_font_size: int = 28
    
    # TTS settings
    tts_enabled: bool = True
    tts_speed: float = 1.0
    
    # Visual settings
    high_contrast: bool = False
    opendyslexic_font: bool = False
    
    # Keyboard settings
    keyboard_navigation: bool = True
    
    def to_dict(self) -> dict:
        return {
            'captions_enabled': self.captions_enabled,
            'caption_font_size': self.caption_font_size,
            'tts_enabled': self.tts_enabled,
            'tts_speed': self.tts_speed,
            'high_contrast': self.high_contrast,
            'opendyslexic_font': self.opendyslexic_font,
            'keyboard_navigation': self.keyboard_navigation,
        }
    
    @classmethod
    def from_dict(cls, data: dict) -> 'AccessibilitySettings':
        return cls(
            captions_enabled=data.get('captions_enabled', True),
            caption_font_size=data.get('caption_font_size', 28),
            tts_enabled=data.get('tts_enabled', True),
            tts_speed=data.get('tts_speed', 1.0),
            high_contrast=data.get('high_contrast', False),
            opendyslexic_font=data.get('opendyslexic_font', False),
            keyboard_navigation=data.get('keyboard_navigation', True),
        )
    
    def toggle_high_contrast(self) -> None:
        """Toggle high contrast mode"""
        self.high_contrast = not self.high_contrast
        
    def enable_high_contrast(self) -> None:
        """Enable high contrast mode"""
        self.high_contrast = True
        
    def disable_high_contrast(self) -> None:
        """Disable high contrast mode"""
        self.high_contrast = False

## Notes & Considerations

### Edge Cases

- **Mid-Game Toggle**: User switches during gameplay - theme should update immediately without resetting
- **Asset Colors**: Sprites/graphics may need recoloring or masks for high contrast
- **Photos/Images**: Any photos would need high contrast filters (not applicable to this game)
- **Overlapping Elements**: High contrast borders may cause clipping - increase padding

### Implementation Tips

1. **Central Theme System**: Use ThemeManager everywhere - don't hardcode colors
2. **CSS-like Variables**: Theme colors accessed by name, not hardcoded
3. **Cache Palettes**: Load color palettes once, reuse
4. **Test Early**: Test high contrast on actual displays, not just IDE

### Accessibility Best Practices

- **Never Use Red/Green Alone**: High contrast color blind safe (cyan instead of red)
- **Thick Borders**: 4px minimum for high contrast buttons
- **Focus Always Visible**: Never remove outline in high contrast
- **No Gray текста**: Ensure text is white or near-white on black

### Color Choices Explained

| Choice | Reason |
|--------|--------|
| Black background | Maximum contrast (21:1 with white) |
| White text | Maximum contrast (standard) |
| Yellow focus | High visibility, color blind safe |
| Cyan error | Color blind safe alternative to red |
| Green success | Bright enough on black |

### Performance Considerations

- **Theme Switch**: Should be instant (<100ms)
- **Memory**: Minimal (just color palette in memory)
- **Rendering**: No performance impact (just different colors)
